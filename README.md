# SpeakRight

SpeakRight is an AI-powered language learning app designed to help users practice French conversation in real-life scenarios with instant feedback, translation, and audio playback.

---

## Features

- 🧠 AI conversation simulation based on selected scenarios (e.g., job interview, hotel, restaurant)
- 🎧 Converts AI text responses into speech using gTTS
- 🌍 Translates French AI responses into English
- 🔊 Uses AssemblyAI or gTTS for text-to-speech generation
- 💬 Scenarios supported with role-based prompts
- 🌐 Built with Node.js, Express, and TypeScript on the backend

---

## Technologies

- Node.js + Express
- TypeScript
- OpenRouter API (DeepSeek model)
- Google Text-to-Speech (gTTS)
- AssemblyAI (for optional speech features)
- dotenv for managing environment variables
